## GitHub Pages Configuration

The repository is configured for GitHub Pages deployment from the `docs/` directory. The site employs a formal, authoritative design aesthetic consistent with institutional technical documentation standards.

### Deployment Settings
1. Repository Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main` (or your default branch)
4. Folder: `/docs`
5. Custom domain (optional): `www.criminalwebservices.com`
